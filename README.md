# Facial Emotion Recognition using OpenCV and FER

## Overview
This project detects human emotions in real-time from the webcam using OpenCV and the FER (Facial Emotion Recognition) library.  
The model identifies emotions such as happy, sad, angry, surprise, neutral, and more.

---

## Features
- Real-time face detection using webcam  
- Emotion classification for detected faces  
- Display emotion labels and probabilities on the video frame  
- Draw bounding boxes around faces  

---

## Tech Stack
- Python  
- OpenCV  
- FER (Facial Emotion Recognition library)  

---

## How to Run
1. Install dependencies by running:  
pip install -r requirements.txt

2. Run the project by executing:  
python main.py

3. Press **Esc** to exit the webcam feed.
